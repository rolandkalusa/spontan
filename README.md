# SPONTAN — Improv in Vienna

A community website prototype for Vienna's improvised theater scene.

## What this is

SPONTAN is a single-page prototype for discovering improv events, venues, and resources in Vienna. It's built as a static HTML file with no dependencies — just open `index.html` in a browser.

## Features

- **Events** — Rolling 7-day event feed with load more/past, filterable by type (jam, show, workshop), language (EN/DE), and a persistent beginner-friendly toggle
- **News** — Community news cards
- **Rooms** — Rehearsal and performance space directory with capacity, pricing, and equipment info
- **Start Improv** — Step-by-step guide for newcomers
- **Event details** — Full event pages with Google Maps embed and add-to-calendar (Google Calendar / .ics download)
- **Bilingual** — Full EN/DE language toggle
- **Dark mode** — System-aware with manual toggle
- **Newsletter signup** — Email capture card (frontend only, no backend)

## Status

This is a **clickable prototype** with sample data. Event data is hardcoded and repeats weekly. The next step is migrating to a Next.js site with real event data from Google Calendar.
